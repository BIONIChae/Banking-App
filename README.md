# UR Banking-App

A beginner-friendly banking application built with C# in Visual Studio.
This project simulates the core features of a digital banking system while demonstrating GUI design, object-oriented programming, and user interaction in C#.

<h5>Project Overview</h5>

The goal of this project is to demonstrate:
- C# GUI development in Visual Studio
- Object-Oriented Programming (OOP) principles
- Event-driven programming with buttons, forms, and input fields

<h5>Features</h5>

✔ Create a bank account
✔ View account details
✔ Deposit money
✔ Withdraw money
✔ Check account balance
✔ Simple and intuitive GUI

<h5>Technologies Used</h5>

C#
.NET Framework / .NET
Windows Forms (WinForms) or WPF (depending on your project type)
Visual Studio

<h5>Concepts Demonstrated</h5>

Classes and Objects
Encapsulation
Inheritance
Polymorphism
Constructors
Methods and Event Handlers
GUI Components
User Input Validation
